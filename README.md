# QR Code Generator

A simple QR Code generator using a single HTML file. To run this open the HTML file in your browser.

# Features
- Lightweight website to generate QR code for given string.

# Usage
1. Go to https://omrpatel.github.io/simple-qrcode-generator/.
2. Enter the website URL in input text.
3. Click on the "Generate" button.
4. Scan the generated QR code.
